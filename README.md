# Smart-Flight-Booking-Automation
Automates real-world flight booking by dynamically analyzing and selecting the lowest-priced option at runtime. Built using a data-driven framework with POM to simulate intelligent user decision-making.

# ✈️ Smart Flight Booking Automation

### Data-Driven Test Framework with Dynamic Price Selection

---

## 🌟 Project Vision

Most automation projects stop at simple form submission or basic validations.
This project goes beyond that.

💡 The goal was to simulate how a real user searches and books flights — while also making smart decisions based on available data.

👉 Instead of selecting a predefined option, the framework dynamically analyzes flight prices and chooses the most cost-effective option.

---

## 🎬 What This Project Demonstrates

🧑‍💻 Imagine a real user doing this:

✔ Searches for flights between cities
✔ Views multiple available options
✔ Compares ticket prices
✔ Selects the cheapest flight
✔ Enters booking details
✔ Completes the booking successfully

🎯 This framework automates the entire decision-making and booking flow seamlessly

---

## 🧠 Smart Automation Logic

This project is not just automation — it includes **dynamic decision-making**.

💡 At runtime, the framework:

* Captures all available flight prices
* Compares them programmatically
* Identifies the lowest price
* Selects the corresponding flight automatically

👉 This mimics real-world user behavior instead of static test execution.

---

## 🏗️ Framework Design (Industry Standard)

🔹 **Page Object Model (POM)**
Keeps locators and actions separated
Improves readability, reusability, and maintainability

🔹 **Data Driven Testing (TestNG)**
Executes the same flow across multiple datasets
Covers multiple routes without duplicating code

🔹 **TestNG Execution Flow**
Manages test lifecycle using annotations
Ensures clean setup and teardown

---

## 🔄 End-to-End Test Flow

👉 The automation follows a complete user journey:

1. Launch application
2. Select departure and destination cities
3. Search for available flights
4. Extract and compare all prices
5. Select the cheapest flight dynamically
6. Fill booking details
7. Confirm booking
8. Repeat for multiple datasets
